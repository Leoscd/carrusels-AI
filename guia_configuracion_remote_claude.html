Configuración de
Remote Claude Code

Esta guía detalla el proceso para habilitar el acceso remoto a tu instancia de Claude Code, permitiéndote ejecutar análisis complejos, presupuestos y consultas normativas directamente desde la obra, utilizando la potencia de tu estación de trabajo principal.

Requisitos Previos

• Claude Code instalado en tu PC (v2.4 o superior).
• Conexión a internet activa en ambos dispositivos.
• Sesión de Anthropic configurada.

Paso a Paso
PASO 01

Iniciación en Estación Base

Abre tu terminal favorita (VS Code, Cursor o Terminal de sistema) en la PC donde tienes almacenada tu documentación de obra y bases de precios.

PASO 02

Habilitar túnel remoto

Dentro de la terminal de Claude Code, ejecuta el comando /remote. El sistema generará una URL segura y un código de vinculación temporal.

PASO 03

Vinculación Móvil

Escanea el código QR o ingresa la URL en tu smartphone. A partir de este momento, cualquier instrucción que des desde tu teléfono será procesada por tu PC principal.

ARQUITECTURA & IA

Para maximizar resultados en obra, asegúrate de que Claude tenga acceso a tu carpeta de "Precios de Referencia" y "Ordenanzas Municipales" antes de salir. Esto te permitirá pedir presupuestos exactos en tiempo real frente al cliente.
